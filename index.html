How to put your website on Github!

Step 1

After right clicking on your mouse in the file with the code and clicking git bash. In the consule you will type "Git Init"

Step 2

Next, you will change the branch by typing git checkout-b gh-pages.

Step 3

To make sure you have did this correctly, type "git status" and see if your html/css files are present.

Step 4

After you complete those tasks, you will need to goto the Github website and log in, once you are logged in. In the upper corner you will need to click on New repository.

Step 5

Here is whee you will name your new website, once you have typed its new name you may click create repository.

Step 6

Now, that you are done with that, go back into the console and log in with the commands git config --global user.name (type your username) and git config --global user.email (your email)

Step 7

After that to make sure you are on the right track type "git status" to make sure everything is happening how it is supposed to, when both files names pop up, you will need to add them both, to do this you type "git add" and the file name exactly how it is written in git status.

Step 8

Now type "git status" again and if your files are green you are doing good! Now you can commit them using "git commit -m "message""

Step 9

Go back on the Github site these links should be on your page, you are going to want to copy the first one.

Step 10

Now paste it in with "insert key" and hit enter. Next, type "git push -u gh-pages" then it will ask you to log in to your site, it will not show you typing yor password so do not freak out that it is not working! If you do mess up yor password just type the command again and do it again.

Finish

You are done, and if you go back to your profile you can see your repositories. To view your website go to yourusername.github.io/yoursitename.
